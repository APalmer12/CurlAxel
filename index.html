CurlAxel by JokerHacker

PHP based download accelerator

this is a php class made to be integrated in php scripts. It uses curl to download files using multiple parallel connections in order to get the highest speed available.

Depdendencies

php php_curl apache jquery (for animated progress bars)

Install

it's a php class, just call it in your script, see readme for more infos
